#  Cardápio de Hamburgueria Fictício

Este é um projeto front-end de um cardápio interativo para uma hamburgueria fictícia. O objetivo foi desenvolver uma interface moderna, responsiva e funcional, utilizando HTML, Tailwind CSS e JavaScript puro.

---

##  Tecnologias Utilizadas

- **HTML5**
- **Tailwind CSS** – estilização rápida e responsiva
- **JavaScript** – manipulação de elementos, interatividade e lógica do cardápio

---

##  Funcionalidades

- Exibição de produtos organizados por categorias
- Layout responsivo para mobile e desktop
- Design moderno com foco em usabilidade
- Estrutura fácil de adaptar para uso real
